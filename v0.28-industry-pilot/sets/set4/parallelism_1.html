Problem Text

Our client requested that our system include more functions that would upgrade it. We managed to add these use cases, despite previous concerns about meeting our deadline and handle code defects discovered earlier.

Option A

“meeting our deadline and handling code defects”

This option uses consistent action words such as “meeting” and “handling.”

Option B

“planning to meet our deadline and handling code defects”

This option pairs two action words, “planning” with “handling,” but the “planning” portion is really the phrase, “planning to meet,” which is not consistent with the word “handling” alone.”

Option C

“meeting our deadline and code defects”

Unlike the first two options, this pairs an action word with a thing: “meeting” with “code defects”; therefore, these two related words are not consistent.
