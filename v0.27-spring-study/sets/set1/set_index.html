Cover Letter
The following section presents excerpts from a student's cover letter for a Web Application Development position at Nintendo of America. The student hopes to impress recruiters with his experience, which, he believes, fulfil the job requirements. However, the recruiters did not find the cover letter convincing enough. In the challenges that follow, try to consider the options presented so that the student can make the most effective case for the audience.

Directions: Identify and improve text that can help the student use coherence, concision, and correctness more effectively.
